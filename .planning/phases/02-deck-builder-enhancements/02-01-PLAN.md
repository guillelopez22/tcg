    2. Add to `deck.service.ts` — import `deckShareCodes` from `@la-grieta/db`, import `autoDetectAndParse` from `@la-grieta/shared`, import `customAlphabet` from `nanoid`:

    3. Add `generateShareCode(userId: string, deckId: string): Promise<string>`:
       - Query deck by id, verify ownership (deck.userId === userId), verify isPublic === true
       - If not owner: throw TRPCError FORBIDDEN
       - If not public: throw TRPCError FORBIDDEN with message "Only public decks can be shared. Make this deck public first."
       - Generate code: `LG-` prefix + 6 chars from customAlphabet('ABCDEFGHJKLMNPQRSTUVWXYZ23456789', 6)
       - Insert into deckShareCodes, retry up to 3 times on unique violation
       - Return the code string

    4. Add `resolveShareCode(code: string): Promise<DeckWithCards>`:
       - Query deckShareCodes WHERE code = input, join to get deckId
       - If no row: throw TRPCError NOT_FOUND "Deck no longer available"
       - Call existing getById(null, { id: deckId }) to get full deck with cards
       - Return the deck

    5. Add `importFromText(userId: string, input: { text: string; name?: string }): Promise<{ resolved: DeckCardEntry[]; unmatched: string[]; deckName: string }>`:
       - Call autoDetectAndParse(input.text) to get ParseResult
       - For each entry in ParseResult.entries, resolve card names against cards table using ILIKE on cleanName column (case-insensitive partial match)
       - Return { resolved entries (cardId + quantity + zone), unmatched names, deckName: input.name || 'Imported Deck' }
       - Do NOT create the deck yet -- the client shows a preview first, then calls deck.create with the resolved entries

    6. Add `importFromUrl(userId: string, input: { url: string; name?: string }): Promise<{ resolved: DeckCardEntry[]; unmatched: string[]; deckName: string }>`:
       - Server-side fetch of the URL with 10-second timeout using AbortController
       - Extract text content from the HTML response (strip HTML tags, get text)
       - Pass extracted text to importFromText logic
       - On fetch error or timeout: throw TRPCError BAD_REQUEST "Could not fetch that URL. Try pasting the deck list text instead."

    7. Extend `browse()` method to handle the new `championName` filter:
       - When input.championName is provided, add a WHERE condition using a subquery: deck ID must be IN (SELECT dc.deck_id FROM deck_cards dc INNER JOIN cards c ON dc.card_id = c.id WHERE dc.zone = 'champion' AND c.clean_name ILIKE '%{escapedName}%')
       - Use the existing `escapeLike` utility from @la-grieta/shared for safe ILIKE input

    8. Add new endpoints to `deck.router.ts`:
       - `shareCode` sub-router or flat endpoints:
         - `generateShareCode`: protected procedure, input shareCodeGenerateSchema, calls deckService.generateShareCode
         - `resolveShareCode`: public procedure (optionalAuth), input shareCodeResolveSchema, calls deckService.resolveShareCode
       - `importFromText`: protected procedure, input deckImportTextSchema, calls deckService.importFromText
       - `importFromUrl`: protected procedure, input deckImportUrlSchema, calls deckService.importFromUrl
       - Import the new schemas from @la-grieta/shared

    9. No changes needed to deck.module.ts unless new dependencies require injection.

    NOTE: The `browse` endpoint is already public (pub procedure). The championName filter addition is backward-compatible -- existing callers that don't pass championName get the same behavior.
