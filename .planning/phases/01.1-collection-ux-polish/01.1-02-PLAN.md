  <files>apps/web/src/app/(dashboard)/collection/collection-grid.tsx</files>
  <action>
    Modify collection-grid.tsx to add optimistic updates and undo behavior. This is the foundational change before adding long-press and copy picker.

    **Optimistic count tracking:**
    - Add state: `const [optimisticDeltas, setOptimisticDeltas] = useState<Map<string, number>>(new Map())`.
    - In the card grid rendering, compute `displayCount = copies.length + (optimisticDeltas.get(card.id) ?? 0)`.
    - On add mutation `onMutate`: bump delta +1 for that cardId. On `onSettled`: clear delta and invalidate.
    - On remove mutation `onMutate`: bump delta -1 for that cardId. On `onSettled`: clear delta and invalidate.
    - On `onError`: roll back delta and show `toast.error('Failed')`.

    **Undo toast on remove:**
    - Before calling the remove mutation, capture the removed copy's `{ id, cardId, variant, condition }`.
    - After remove mutation succeeds (`onSuccess`), show sonner toast:
      ```
      const toastId = toast('Copy removed', {
        duration: 5000,
        action: {
          label: 'Undo',
          onClick: () => {
            addMutation.mutate({ cardId, variant: removedCopy.variant, condition: removedCopy.condition });
            toast.dismiss(toastId);
          },
        },
      });
      ```
    - The undo re-adds with the exact variant and condition of the removed copy.

    **Total value badge:**
    - After Plan 01, each entry's `card` object has `marketPrice` and `foilMarketPrice`.
    - Compute filtered total from currently loaded pages:
      ```
      const filteredTotal = useMemo(() => {
        if (!data?.pages) return 0;
        return data.pages.flatMap(p => p.items).reduce((sum, entry) => {
          const price = isFoilVariant(entry.variant)
            ? parseFloat(entry.card.foilMarketPrice ?? '0')
            : parseFloat(entry.card.marketPrice ?? '0');
          return sum + price;
        }, 0);
      }, [data?.pages]);
      ```
    - Import `isFoilVariant` from `@la-grieta/shared`.
    - Display in the filter/sort bar area: `<span className="text-zinc-400 text-sm tabular-nums">Total: ${filteredTotal.toFixed(2)}</span>`.

    - Ensure the card grid item wrapper has `group` class for Tailwind group-hover.
