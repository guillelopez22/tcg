**1. [Rule 1 - Bug] Stale compiled .js/.d.ts files in packages/shared shadowed new constants**

- **Files modified:** packages/shared/src/constants/card.constants.js, packages/shared/src/constants/card.constants.d.ts, packages/shared/src/schemas/deck.schema.js, packages/shared/src/schemas/wishlist.schema.js, packages/shared/src/utils/validate-deck-format.js, packages/shared/src/index.js
- **Commit:** 9a637c0

**2. [Rule 1 - Bug] Test mock missing validateCardIdsExist DB select slot**
- **Found during:** Task 1 (TDD GREEN — "This deck is private" error in tests with create + cards)
- **Issue:** `create()` calls `validateCardIdsExist` (1 DB select) then `buildCardTypeMap` (1 DB select). Tests that only pushed `buildCardTypeMap` select were misaligned — the wrong data landed in each slot.
- **Fix:** Added explicit `validateCardIdsExist` select push before `buildCardTypeMap` push in all affected test cases
- **Files modified:** apps/api/__tests__/deck.service.spec.ts
- **Commit:** 9a637c0
